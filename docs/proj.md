# Making Particle Transport More Memory-Efficient: Reduced Memory Space-Time Discretization Schemes
This project has focused on the development of space-time discretization schemes with reduced memory for solving the linear Boltzmann transport equation, the equation that describes how very small particles move and interact with a medium. These schemes are based on the backward-Euler time-integration scheme and the (bi/tri)linear-discontinuous finite element method. By "throwing out" high-dimensional grid functions between timesteps, and reconstructing them as needed, this project has reduced the persistent memory allocation requirement for solving particle transport problems by (up to - depending on implementation) a factor of 87.5% in 3D simulations. Collaborators on this project include professors from North Carolina State University and Texas A&M University, as well as Staff Scientists at Los Alamos National Laboratory. <br>

# Sweating the Small Stuff: Efficient Low-Order Solves for Particle Transport Acceleration Methods
Efficient and accurate solutions to the radiation diffusion equation are important at all levels of deterministic particle transport. Whether the diffusion equation is being used to model the particle distribution, or as a preconditioner for your transport solve, being able to get accurate solutions in the shortest-possible amount of time is extremely important. When used as a preconditioner, or within an acceleration method, it is important that the diffusion equation discretization is compatible with the discretization of the transport equation - a difficult task when using discontinuous finite elements. This work has focused on exploring interior penalty methods to discretize the diffusion equation for use in diffusion synthetic acceleration and linear-multifrequency gray acceleration techniques, primarily looking at the computational efficiency of the low-order (inner) solves in relation to the number of transport (outer) iterations. This project is being performed in collaboration with Staff Scientists at Los Alamos National Laboratory. <br>

# Splitting Cells Without Losing Count: Adaptive Mesh Refinement Remapping for Corner-Centered Values
When running large physics simulations, it's often useful to dynamically refine the computational mesh by splitting coarse grid cells into finer ones in regions where more detail is needed. This process, called adaptive mesh refinement (AMR) is an important tool for improving simulation accuracy without excessive computational cost. The tricky part is making sure that when you split a cell, you don't lose track of the physics. For quantities like energy or particle count, you want the refinement process to be *conservative*, i.e. you want the total amount of a quantity to remain the same before and after the cell is refined. For simple cell-averaged quantities, this is straightforward. But when values live on the corners or vertices of the mesh rather than the cell centers, the standard approaches become complicated and expensive. This project developed simpler, cheaper methods for remapping corner-centered values during mesh refinement, while still guaranteeing conservation. The result is a cleaner, more efficient refinement process that doesn't sacrifice accuracy, making large-scale simulations faster and easier to work with. This work was done in collaboration with Staff Scientists at Los Alamos National Laboratory.<br>
